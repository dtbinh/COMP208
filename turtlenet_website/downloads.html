Downloads

Want Turtlenet? Get Turtlenet.

The Turtlenet client runs in the Java Virtual Machine meaning that it can be used on any platform with an up to date version of Java. The client also requires a modern web browser such as Google Chrome or Mozilla Firefox.

You can download the latest version of Java HERE.

Turtlenet Client Turtlenet Client (With Tor)
turtlenet.jar turtlenet_tor.jar

N.B. If you plan to use the Tor client you will need to install Tor seperatly. Tor will set up a SOCKS proxy on port 9050 while it's running.

N.B. The Tor client doesn't offer any more security, it's only use is if the use of services such as Turtlenet is prohibited or you have some other reason for hiding that you are using Turtlenet.
