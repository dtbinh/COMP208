long ballmerpeak.turtlenet.server.NetworkConnection.getTime ( )

Get the server time.

Millisecond timestamps can, and have, been used to identify users. We therefore recomend always using server time. (Obviously other methods are in place to hide true network latency.) Thread safe.

Returns
The number of milliseconds since midnight january first 1970, according to the server.
