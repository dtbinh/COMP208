Want Turtlenet? Get Turtlenet.

Turtlenet is currently available on Windows, Linux, and OSX platforms. The latest versions of all variants are available for download currently.

The client, once installed, will run through Chrome or Firefox natively. The server runs through the Java Virtual Machine, as per such any any OS with java can natively run it. There is also a link below to the Sun Microsystems website for those that need the Java Virtual Machine to run the server.
